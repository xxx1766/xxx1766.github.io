反问环节

参考：https://github.com/perkfly/reverse-interview-zh

大类可以分为岗位职责与期望、技术栈与工程实践、 团队协作与日常环境、公司发展与企业文化、薪酬福利与个人成长。
可以细化为职责、技术、团队/同事、公司、社会问题/热点、冲突处理、商业、工作模式、待遇、休假、福利、人才培养等方面

从中挑选 2 到 3 个自己当前最看重的方面（例如如果是冲着技术成长去的，就重点问“技术与工程”和“人才培养”；如果是寻求稳定，就重点问“商业”和“团队协作”），每个大类准备 1-2 个犀利的问题即可。

第一个勾选框表示找工作面试可问；第二个勾选框表示实习面试可问。

岗位职责与期望

  • 职责
    • 日常工作是什么？典型的一天或者一周的工作是怎样安排的？每天预期/核心工作时间是多少小时？
    • 有给我设定什么特定目标吗？预计目标完成度？期望最初的xxx个月能够完成什么？
    • 入职培训什么样？
    • 每个开发者有多大的自由来做出决定？
    • 会怎么样衡量我的绩效？
    • 对我的申请你有什么疑虑么？
    • 在这份工作上，我将会和谁紧密合作？直接上级他们的上级都是什么样的管理风格？
    • 我在这个岗位上应该如何发展？会有哪些机会？
    • 我入职的岗位是新增还是接替之前离职的同事？（是否有技术债需要还）？
    • 入职之后在哪个项目组，项目是新成立还是已有的？
  • 终极问题
    • 该职位为何会空缺？
    • 公司如何保证人才不流失？
    • 这份工作/团队/公司最好和最坏的方面是？
    • 你最开始为什么选择了这家公司？为什么留在这家公司？

技术栈与工程实践

  • 技术
    • 常用的技术栈是什么？介绍一下技术原则或者要求。代码有开发文档吗？
    • 怎么测试代码？怎么追踪 bug？怎样监控项目？怎么集成和部署代码改动？
    • 从计划到完成一项任务的工作流是什么样的？
    • 如何准备故障恢复？
    • 有标准的开发环境吗？是强制的吗？
    • 需要花费多长时间来给产品搭建一个本地测试环境？（分钟 / 小时 / 天）
    • 需要花费多长时间来响应代码或者依赖中的安全问题？
    • 如何管理内部和外部的数字资产？如何管理依赖？
    • 是否有技术分享交流活动？有的话，多久一次呢？
    • 业务需求有没有文档记录？是如何记录的？
    • 是如何面对和解决技术债的？是否有专门的时间或者预算用于重构？
    • 如何进行单元测试呢，是否都有单元测试的习惯？

团队协作与日常环境

  • 团队
    • 工作是怎么组织的？团队有多大？技术团队的架构和人员组成？
    • 团队内 / 团队间的交流通常是怎样的？
    • 如果遇到不同的意见怎样处理？谁来设定优先级 / 计划？
    • 如果团队没能赶上预期发布日期怎么办？
    • 每周都会开什么类型的会议？会有定期的和上级的一对一谈话吗？
    • 产品 / 服务的规划是什么样的？（周一发布/持续部署/多个发布流/…)
    • 生产环境发生事故了怎么办？是否有不批评人而分析问题的文化？
    • 有没有一些团队正在经历还尚待解决的挑战？
    • 如何跟踪进度？
    • 预期和目标是如何设定的？谁来设定？
    • Code Review 如何实施？
    • 如何平衡技术和商业目标？
    • 如何共享知识？
    • 团队内开发、产品、运营哪一方是需求的主要提出方？哪一方更强势？
  • 问未来的同事
    • 开发者倾向于从哪里学习？组里如何组织学习技术交流？
    • 你对在这里工作最满意的地方是？最不满意的呢？如果可以的话，想改变哪里？
    • 团队最老的成员在这里多久了？
  • 远程工作
    • 远程工作和办公室工作的比例是多少？
    • 公司提供硬件吗？更新计划如何？
    • 使用自己的硬件办公可以吗？现在有政策吗？
    • 有共享办公或者上网的预算吗？
    • 多久需要去一次办公室？
    • 公司的会议室是否一直是视频会议就绪的？
  • 办公室布局
    • 办公室的布局如何？（开放的/小隔间/独立办公室）
    • 有没有支持/市场/其他需要大量打电话的团队在我的团队旁边办公？
  • 冲突
    • 不同的意见如何处理？
    • 如果被退回了会怎样？（“这个在预计的时间内做不完”）
    • 当团队有压力并且在超负荷工作的时候怎么处理？
    • 如果有人注意到了在流程或者技术等其他方面又改进的地方，怎么办？
    • 当管理层的预期和工程师的绩效之间有差距的时候如何处理？
    • 如果在公司内你的同事因涉嫌性侵犯他人而被调查，请问你会如何处理？
    • 假设我自己很不幸是在公司内被性侵的受害者，在公司内部有没有争取合法权益的渠道？

公司发展与企业文化

  • 公司
    • 公司为什么在招人？（产品发展/新产品/波动…)
    • 晋升流程是怎样的？要求 / 预期是怎样沟通的？
    • 绩效评估流程是怎样的？
    • 技术和管理两条职业路径是分开的吗？
    • 对于多元化招聘的现状或者观点是什么？
    • 公司的成熟度如何？（早期寻找方向 / 有内容的工作 / 维护中 / …)
    • 我可以为开源项目做贡献吗？是否需要审批？
    • 你认为公司未来五年或者十年会发展成什么样子？
    • 公司的大多数员工是如何看待整洁代码的？
    • 在这里成功的定义是什么？如何衡量成功？
    • 有体育活动或者团建么？
    • 公司支持开源项目吗？
    • 有竞业限制或者保密协议需要签吗？
    • 你们认为公司文化中的空白是什么？
    • 若公司处于不良情况，以及如何处理的故事吗？
    • 您在这工作了多久了？您觉得体验如何？
    • 公司的调薪制度是如何的？
    • 公司有没有申请调岗的制度？
    • 公司对于员工的心理健康和福祉有什么具体措施？
    • 你对在这里工作最满意的地方是？你为什么留在这家公司？
  • 商业
    • 你们现在盈利吗？如果没有的话，还需要多久？如果有的话，年度营业额是大概有多少？
    • 公司的资金来源是什么？谁影响或者制定高层计划或方向？
    • 你们如何挣钱？什么阻止了你们挣更多的钱？
    • 公司未来一年的增长计划怎样？五年呢？公司未来的商业规划是怎样的？
    • 你们的竞争优势是什么？
    • 都在做副业吗？
  • 社会问题
    • 关于多元化招聘/某技术热点什么看法？
    • 这里的工作生活平衡地怎么样？

薪酬福利与个人成长

  • 待遇
    • 如果有奖金计划的话，奖金如何分配？
    • 如果有奖金计划的话，过去的几年里通常会发百分之多少的奖金？
    • 有五险一金(zh)/401k(us)或者其他退休养老金等福利吗？
    • 五险一金中，补充公积金一般交多少比例？/401k一般交多少比例？我可以自己选择这一比例吗？
    • 有什么医疗保险吗？如果有的话何时开始？
    • 有额外商业保险吗？例如人寿保险和额外的养老/医疗保险？
    • 商业保险可以给家人办理吗？成年人/未成年人？
    • 是否可以申请更换工作地点？
    • 是否愿意协助海外应聘者申请工作签证？
  • 休假
    • 带薪休假时间有多久？
    • 病假和事假是分开的还是一起算？
    • 我可以提前使用假期时间吗？也就是说应休假期是负的？
    • 假期的更新策略是什么样的？也就是说未休的假期能否滚入下一周期
    • 照顾小孩的政策如何？
    • 无薪休假政策是什么样的？
    • 学术性休假政策是怎么样的？
    • 孕产假政策具体是怎样的？
  • 福利
    • 公司提供 mac 开发吗？使用自带电脑有补贴吗？
    • 公积金多少比例缴纳？
    • 公司是否有食堂，是否有餐饮福利补贴？是否提供租房补贴？是否提供话费补贴？是否有交通补贴？
  • 人才培养
    • 升职加薪条件是否量化?
    • 每年给团队安排多少费用用于学习培训?

    每年组织多少次关于技术能力提升的讲座/论坛？
